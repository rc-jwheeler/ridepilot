Accessing RidePilot

RidePilot is implemented as a web-based application that provides an enhanced user experience with cross-browser compatibility.

To access RidePilot, you need:

·         A desktop computer, tablet or mobile device

·         An internet connection

·         A HTML5-compatible internet web browser such as:

For best performance and compatibility, it is recommended that you upgrade to the latest version of your web browser.

Login

For the UTA deployment, RidePilot can be accessed at https://ridepilot.rideuta.com/.

Once the page loads, you will be prompted to login. Enter your email and password and click Log In.

If you cannot remember your password, click Forgot your password? and you will be emailed details on how to reset it.

User Interface

The basic controls of the RidePilot application appear on each page:

·         Title bar containing the name of the application, the name of the User who is currently logged in, and the name of the selected Provider (typically the Provider that the User is affiliated with).

·         A series of tabs - Trips, Runs, etc. - that display the various pages within the application. Each of these tabs is described below.

·         Page body that changes based upon the selected tab.

·         Page footer indicating the version number.

Logout

To log out of RidePilot, either:

·         Close the web browser, or

·         Click the Log out link in the title bar.
